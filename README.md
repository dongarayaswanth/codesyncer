# Code Syncer

A hacker-themed web application that allows you to save, retrieve, and manage code snippets directly on GitHub.

##  Live Demo
**[Open Code Syncer](https://dongarayaswanth.github.io/codesyncer/)**

##  Features

- **GitHub Integration**: Saves your code snippets directly to your GitHub repository.
- **Perfect Formatting**: Preserves all spaces, tabs, and indentation.
- **Hacker Themes**: Choose from Green (Matrix), Orange (Retro), Blue (Cyber), Dark, White, and Black themes.
- **Search**: Instantly filter your saved snippets.
- **Edit & Delete**: Manage your files directly from the interface.
- **Secure**: Your Personal Access Token is stored locally in your browser.

##  Setup

1. **Fork/Clone** this repository.
2. Enable **GitHub Pages** in Settings -> Pages (Source: \main\ branch).
3. Open the live link.
4. Click the **Key Icon ()** and enter your GitHub Personal Access Token (with \epo\ scope).
